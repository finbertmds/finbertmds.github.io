### 💸 グループ割り勘アプリ

グループの費用精算をすばやく簡単に！旅行、パーティー、BBQ、同窓会などに最適。**walica.jp**を参考にしつつ、モバイル向けに改良。

**主な特徴：**

- イベント名、メンバー、通貨で素早く作成 💰  
- LINEやQRコード、メールで招待共有 🔗  
- すべてのメンバーが支出記録可能 🧾  
- 「誰が誰にいくら」かを最小限の回数で計算 🤝  
- 複数通貨対応、換算も自動 🌐  
- モダンなUI、ライト/ダークモード対応 🌙☀️  
- CSV出力＆簡単シェア 🧮  
- （近日公開）クラウド同期、Google/Appleログイン ☁️  
- 支払済みチェック、リマインダー、通知 🔔
